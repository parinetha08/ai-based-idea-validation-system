# Demo Flow - AI Idea Validator

## Goal
Show how startup ideas can be validated in seconds using AI.

---

## Step-by-step Demo

### 1. Introduction (10 sec)
“We built an AI system that validates startup ideas like an investor.”

---

### 2. Input Demo (20 sec)
- Open Streamlit UI
- Enter idea:
  "AI app for farmers to predict crop diseases"

---

### 3. Processing (10 sec)
- Click Analyze
- Show loading animation (AI thinking)

---

### 4. Output Display (40 sec)
Show:
- Idea Score (0–100)
- Market Demand
- Risks
- Improvements
- Similar ideas

Highlight score visualization.

---

### 5. Explain Logic (20 sec)
- Mention scoring system
- AI + rule-based hybrid model
- Structured JSON output

---

### 6. Startup Pitch (20 sec)
“This can become a SaaS product for founders to validate ideas before building.”

---

## Bonus (if time allows)
- Compare 2 ideas
- Show improvement suggestions feature
