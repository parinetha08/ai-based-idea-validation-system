# Idea Validation System

## Overview
This system validates startup or project ideas using AI. It analyzes feasibility, market potential, innovation level, and trends.

## Goals
- Evaluate user-submitted ideas
- Provide a numeric score (0–100)
- Give improvement suggestions
- Support multilingual input (English, Telugu, Hindi)

## Features
- AI-based idea analysis (Ollama / backend model)
- Trend checking module
- Scoring engine
- Streamlit UI dashboard
- Multilingual support (i18n)

## Output
- Score breakdown
- Strengths & weaknesses
- Market insights
- Suggestions for improvement
