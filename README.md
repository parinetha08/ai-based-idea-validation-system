# AI Idea Validator 🚀

## What it does
Validates startup ideas using AI and gives:
- score
- risks
- improvements
- market demand

## Tech Stack
- Streamlit
- Python
- OpenAI API
- FastAPI (optional)

## Features
- Real-time idea scoring
- AI feedback engine
- Startup suggestion generator

## How to run
pip install -r requirements.txt
streamlit run app/frontend/streamlit_app.py
